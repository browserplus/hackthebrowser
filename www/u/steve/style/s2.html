Welcome to the YUI Project

The YUI Project comprises a family of tools — including JavaScript, CSS, and developer tools — that empower web developers to build fast, performant projects.

  • YUI Library JavaScript and CSS:
    • YUI 2.x codeline (mature, production-quality code)
    • YUI 3.x codeline (preview release available)
  • YUI Compressor — minify and obfuscate your JavaScript and CSS
  • YUI Doc — generate beautiful, organized, searchable API docs for your JS code

Contribute to YUI

Anyone can contribute to YUI by signing and returning a Contributor's Licence Agreement. Visit the YUI Contributor's Page for more information.

About YUILibrary.com

This site will become the principal online home for the YUI Library over the coming months. Today, the majority of documentation for YUI can be found on the Yahoo Developer Network. If you're looking to get started with YUI, please begin with the YUI website on YDN.
